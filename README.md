# IJARA Transportation LLC — Official Website

Professional truck dispatching website built with React + Vite + Tailwind CSS.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ installed
- npm or yarn

### Installation & Run

```bash
# 1. Navigate to the project folder
cd ijara-transport

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev

# Open http://localhost:5173 in your browser
```

### Build for Production

```bash
npm run build
npm run preview
```
